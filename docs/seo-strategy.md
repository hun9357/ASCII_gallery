### Meta Tag Best Practices

**Title Tag Rules:**
1. **Length:** 50-60 characters (optimal for mobile)
2. **Format:** Primary Keyword | Secondary Keyword | Brand
3. **Include numbers:** "200+", "50+", "Top 10" improve CTR
4. **Action words:** "Copy", "Paste", "Free", "Instant"

**Description Tag Rules:**
1. **Length:** 150-160 characters
2. **Include CTA:** "Copy instantly", "Free forever"
3. **Keywords:** Natural placement of 2-3 target keywords
4. **Benefit-focused:** What user gets, not what site has

**Keywords Meta Tag:**
- Not used by Google for ranking BUT helps organize your strategy
- Include 5-7 relevant keywords
- Use for internal documentation

### Duplicate Content Prevention

**Potential Issues:**

1. **Search Results Pages**
   - Issue: `/?q=cat` creates infinite indexable pages
   - Solution: Add `<meta name="robots" content="noindex">` to search result pages
   - Better: Use Astro middleware to detect query params

2. **Tag Filter Pages**
   - Issue: `/?tags=cute` vs `/?tags=simple` vs `/?tags=cute,simple`
   - Solution: Canonical all filtered views back to main page

3. **Category + Tag Combinations**
   - Issue: `/category/animals?tags=cute` duplicates content
   - Solution: Canonical to category page, add `rel="nofollow"` to tag links

4. **Case Sensitivity**
   - Netlify is case-sensitive by default
   - Solution: Force lowercase redirects in netlify.toml

```toml
[[redirects]]
  from = "/art/*"
  to = "/art/:splat"
  status = 301
  force = false
  conditions = {Language = ["en"]}
```
